# AsaHome-Service

AsaHome-Service provides decrypt_tools for xp3 files of AsaProject games, and the TTS / WindowPet source codes in future.

## Structure

```
AsaHome-TTS/
├── decrypt_tools/              #   XP3 decrypt_tools for AsaProject's Game 
│   ├── decrypt_all.py             
│   ├── decrypt_voice.py           
│   ├── extract_voice_text.py   #   TTS-DataMaker 
│   ├── extract_keys.py            
│   └── README.md                  
│
├── docs/
│   ├── experience.md           #   Experience of AI-Supported Reverse Engineering Process
│   └── report.md               #   A readable report for the decrypt methods.
|
├── TTS/                        #   TTS-Train-Source-Code (Remain to be done)
|
|
├── WindowPet/                  #   WindowPet Source-Code (Remain to be done)
│
├── LICENSE
└── README.md
```
